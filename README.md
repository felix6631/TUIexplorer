# TUIexplorer
TUIexplorer is TUI program to emplement explorer.exe of Windows in TUI, multi-platform.
*** Built with Only C! ***

## What's on left side
On left side, there's tree structure of directory, from home to current directory.

## What's on right side
On right side, there's list of files, just like "ls -la" command

order is follow:
filename - touch time    -    file size - owner - group  -  permission
file.txt   yyyy/mm/dd hh:mm   3.14 MB     A6      A6group   -rwxrwxrwx
